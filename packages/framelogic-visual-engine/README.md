# FrameLogic Visual Engine

Pure visual-decision intelligence extracted from the strongest reusable concepts in the legacy Remotion renderer. It does not draw pixels and does not import React or Remotion.

It produces JSON-safe direction for any renderer:

- Art direction and palette treatment
- Motion profile and bounded camera behavior
- Media grade, crop, and placement
- Campaign-level layout rhythm without adjacent repetition
- Text frame placement and safe-width decisions

Use an adapter to translate the output into Remotion, FFmpeg, Canvas, native UI, or another renderer.
